boards

Everything a channel log can't hold.

🧵

Threaded, Not Scrolling

Replies nest under a subject instead of vanishing into scrollback. Come back in a week, the context is still there.

🪪

One Account, Both Apps

Log in with the same identity you use on Ray Chad IRC. No second registration, no separate password.

🐳

Docker Ready

Ships with its own docker-compose.yml, or bolt it onto an existing raychd stack with one service block.

🔒

Self-Hosted

Your database, your backups, your moderation rules. Nothing phones home.

📜

Open Source

Apache-2.0, same as raychd. Fork it, re-skin it, run it for one channel or fifty.

🔗

IRC Bridge Bot

Optional bot posts new thread titles into your IRC channels — the forum surfaces itself where people already are.

quick start

Up in 3 commands.

01
# Clone the repo
git clone https://github.com/newssungoldentoday-dev/raychd-forum.git
02
# Start with CD
cd raychd-forum
03
# Or Bash Command
bash pkg-install
03
# Or install manually
bash pkg-install
